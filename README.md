# crisis-api

Backend component of CRISIS Project

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Lint

```bash
# lint
$ npm run lint
# lint fix
$ npm run lint:fix
```
